Otro elemento HTML que te permite estructurar el contenido de la página es el elemento lista. Gracias a él puedes crear filas con información y marcarlas con viñetas o números, incluso, hay navegadores que admiten los números romanos.

Aunque no parezca, detrás de las listas que ves en los sitios web, siempre hay una etiqueta HTML indicando que sea una lista desordenada u ordenada. Aquí te explicamos su principal diferencia.

En HTML, podemos enumerar elementos en forma ordenada o desordenada:

Lista ordenada

Una lista ordenada utiliza números o algún tipo de notación que indica una serie de elementos.

Un elemento de lista ordenada se ve de la siguiente manera:

<ol> </ol>

Pero, a diferencia de los elementos que has visto hasta ahora, estas etiquetas no hacen nada por sí solas. Para mostrar el contenido como una lista, necesitan dos elementos HTML que trabajen juntos: uno para indicar que vas a incluir una lista <ol>, y otro para hacer que cada elemento de la lista aparezca en una fila diferente <li>, así:

<ol>
<li>Este es el primer ítem de la lista.</li>
<li>Este es el segundo ítem de la lista.</li>
<li>Y este el tercero.</li>
</ol>

  1. Este es el primer ítem de la lista.
  2. Este es el segundo ítem de la lista.
  3. Y este el tercero.

Lista desordenada

Un elemento de lista desordenada se ve de la siguiente manera:

<ul> </ul>

Su estructura es muy parecida a la de una lista ordenada, pero aquí, el elemento que indica que vas a incluir una lista es <ul>, mientras que, la etiqueta <li>le asignará a cada elemento una fila diferente:

<ul>
<li>Este es el primer ítem de la lista.</li>
<li>Este es el segundo ítem de la lista.</li>
</ul>

  • Este es el primer ítem de la lista.
  • Este es el segundo ítem de la lista.
