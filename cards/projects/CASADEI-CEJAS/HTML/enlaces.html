Los hipervínculos son una de las innovaciones más interesantes que ofrece la Web. Han formado parte de esta desde el principio, pero hacen que la web sea web.

Mediante estos enlaces, podemos comunicar una página con otra. De esta forma, enlazando documentos HTML podemos acabar tejiendo lo que es Internet.

Para crear un enlace en HTML utilizamos el elemento <a> con la siguiente sintaxis:

<a> Contenido del enlace </a>

Pero solo con esto el enlace no tendrá mucha utilidad ya que el principal objetivo del enlace es enlazar a un destino. Para poder indicar el destino de un enlace utilizamos el atributo href.
En valor del atributo href puede ser cualquier URL que represente un recurso. Es decir, una página, una parte de una página, una URL genérica, un archivo, etc.
De esta forma el enlace en HTML lo crearemos con la sintaxis:

<a href="URL"> Contenido del enlace </a>

Destino del enlace

Pero, ¿dónde se abre el enlace? Por defecto y si no hemos configurado nada en el navegador web que estemos utilizando el enlace se abre en la misma ventana en la que tengamos el enlace.


En el enlace, podemos indicar el destino que queremos darle a dicho enlace. Eso lo podemos hacer mediante el atributo target.
Si queremos que al abrir el enlace, este nos lleve a una nueva pestaña, ajena a la original, debemos utilizar el valor _blank

<a href="enlace.html" target="_blank">
Abrir enlace en una nueva ventana
</a>

Enlaces a una parte del documento

Otra capacidad que tenemos en HTML es la de enlazar a una parte concreta del documento. Imagina que en un documento tenemos un título y varios capítulos. Y lo que queremos hacer desde otro documento HTML o bien desde el mismo documento es enlazar directamente al inicio de un capítulo.


Crear ancla en el documento

Para poder enlazar a una parte concreta de un documento lo primero que tenemos que hacer es marcar esa parte del documento. Para ello contamos con el atributo name. Si creamos un enlace con el atributo name, este enlace se definirá como posición y no como enlace de navegación.
La sintaxis será:


<a name="partedocumento"> Contenido </a>

En este caso el contenido puede ser vacío ya que no se ofrecerá nada para navegar. Es por ello que podemos ponerlo antes de nuestro capítulo.

<a name="cap2"></a><h2>Nuevo Capítulo</h2>
  • Este es el primer ítem de la lista.
  • Este es el segundo ítem de la lista.

Enlazando a una parte del documento

Una vez que hemos creado el ancla del enlace en HTML en un documento es hora de acceder a esa parte del documento. Para ello solo tenemos que poner el nombre que le hayamos dado al atributo name precedido de una almohadilla.
La sintaxis será:

<a name="#cap2">Enlace al capítulo 2</a>
