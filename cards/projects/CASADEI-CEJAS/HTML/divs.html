Divs y Span en HTML

Diferentes entre divs y span.

En HTML, las etiquetas <div> y <span> son elementos contenedores que se utilizan para agrupar otros elementos, pero tienen diferencias clave en cuanto a su comportamiento y uso.

¿Qué son los <div> y <span>?

<div>:

El elemento <div> es un contenedor de bloque. Esto significa que se utiliza para agrupar otros elementos en un bloque, ocupando toda la anchura disponible en la página, y rompe el flujo de la página, haciendo que el contenido posterior aparezca en una nueva línea.

Sus usos comunes son:

  1. Agrupar elementos para aplicar estilos en conjunto.
  2. Crear estructuras de diseño (como contenedores de secciones de una página).
  3. Utilizar en combinación con CSS para crear diseños de página complejos (como layout grids o páginas con varias columnas).

<span>:

El elemento <span> es un contenedor en línea. A diferencia de <div>, no crea un nuevo bloque, sino que se utiliza para agrupar contenido dentro de una línea sin afectar el flujo de la página. Es útil cuando se quiere aplicar estilos a fragmentos pequeños de texto o elementos en línea sin cambiar su disposición en la página.

Sus usos comunes son:

  1. Agrupar una porción de texto o un pequeño bloque de contenido para aplicar estilos.
  2. Usar en conjunto con CSS o JavaScript para modificar el estilo o el comportamiento de partes específicas de un contenido en línea (por ejemplo, colorear una palabra específica dentro de una oración).

Ejemplos de usos

<div>

Ahora que repasamos los conceptos, vamos a dar un ejemplo de los divs y span.
Ejemplo:

<div class="section">
<h2>Encabezado de sección</h2>
<p>Texto relacionado con la sección.</p>
</div>

En este ejemplo, <div> agrupa un encabezado y un párrafo bajo una clase común, lo que facilita la aplicación de estilos con CSS.

<span>:

Ejemplo:

<p>Me gusta <span class="highlight">programar</span> en HTML.</p>

En este caso, el <span> agrupa la palabra "programar" para poder aplicar un estilo específico (como cambiar el color) sin alterar la disposición del texto.
