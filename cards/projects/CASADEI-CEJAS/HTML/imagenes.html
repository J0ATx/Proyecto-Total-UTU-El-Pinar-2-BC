Imagenes en HTML

Todo sobre las imagenes en HTML 5

Las imágenes en HTML se utilizan para insertar gráficos, fotografías u otros elementos visuales en una página web. Estas imágenes pueden mejorar la experiencia del usuario, hacer que la página sea más atractiva y transmitir información de manera más clara o efectiva que solo con texto.

¿Cómo se insertan imágenes en HTML?

Las imágenes en HTML se insertan usando la etiqueta <img>, que es una etiqueta vacía (es decir, no tiene una etiqueta de cierre). La sintaxis básica es la siguiente:

<img src="ruta_de_la_imagen" alt="texto alternativo">

Atributos principales de la etiqueta <img>:

src (source):

Especifica la ruta o URL de la imagen que se desea mostrar. Puede ser una ruta local (como una imagen guardada en tu computadora o servidor) o una URL externa (como una imagen alojada en otro sitio web).
Ejemplo:

<img src="imagenes/foto.jpg" alt="Descripción de la imagen">

alt (texto alternativo):

Es un texto descriptivo que se muestra en caso de que la imagen no se pueda cargar o si un usuario utiliza un lector de pantalla (accesibilidad). Este atributo es importante tanto para la accesibilidad web como para el SEO (optimización en motores de búsqueda).
Ejemplo:

<img src="logo.png" alt="Logo de la empresa">

width y height (dimensiones):

Puedes especificar el tamaño de la imagen en píxeles. Sin embargo, es recomendable no usar estos atributos para redimensionar imágenes de manera significativa, ya que hacerlo podría distorsionar la imagen. Es mejor redimensionar las imágenes antes de subirlas.
Ejemplo:

<img src="foto.jpg" alt="Una foto" width="300" height="200">

title:

Muestra un texto emergente (tooltip) cuando el usuario coloca el cursor sobre la imagen. Es un atributo opcional.
Ejemplo:

<img src="foto.jpg" alt="Una foto" title="Haz clic para ver más">
