La estructura de HTML es la forma en que se organiza y presenta el contenido en un documento web. Se compone de etiquetas que definen diferentes elementos, como el encabezado, el cuerpo y otros componentes. Esta organización es crucial porque permite a los navegadores interpretar y mostrar el contenido correctamente.

Además, una buena estructura juega un papel importante en la optimización para motores de búsqueda , ya que una jerarquía clara ayuda a indexar el contenido. Por último, un código bien estructurado es más fácil de mantener y actualizar, lo que resulta útil a medida que un sitio web crece y evoluciona.

A continuación tendrán ejemplos de como es la estructura basica de HTML (cuando se adquieran más conocimientos sobre el tema, la estructura va a ir variando).

Declaración del DOCTYPE

Esto le dice al navegador que el documento es HTML5.

<!DOCTYPE html>

Etiqueta html

Es el contenedor principal de todo el contenido.

<html lang="es">

Etiqueta head

Contiene metadatos sobre el documento, como el título, la codificación de caracteres y enlaces a hojas de estilo.

<head>

Etiqueta body

Contiene todo el contenido visible de la página, como encabezados, párrafos, imágenes, etc.

<body>

Etiqueta header

Sección que suele incluir el título de la página, logotipos y elementos de navegación. Es una forma de presentar la identidad de la página.

<header>

Etiqueta footer

Sección al final del documento que contiene información adicional. Esto puede incluir: Derechos de autor, Enlaces de contacto, Enlaces a políticas, etc.

<footer>

Estructura

Ahora que repasamos como es la estructura básica de html, vamos a usar todas las etiquetas aprendidas para que puedan observar como quedan todas juntas (si utilizas el codigo proporcionado, no se va a ver nada debido a que no utilizamos ninguna etiqueta de texto).

<!DOCTYPE html>
<html lang="es">
<head>
</head>
<body>
<header>
</header>
<footer>
</footer>
</body>
</html>

¿Porque para la etiqueta head, body, etc utilizamos "/"?

Este simbolo "/" se utiliza para indicar el cierre de una etiqueta.
