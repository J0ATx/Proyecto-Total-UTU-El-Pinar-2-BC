Texto en HTML

Diferentes formas de escribir. Descubrelas a continuación.

El texto en HTML se refiere a cualquier contenido textual que se incluye en un documento HTML. Este texto se estructura mediante etiquetas, que indican cómo se debe presentar y organizar la información.

La principal función del texto en HTML es proporcionar contenido a las páginas web, permitiendo que los navegadores interpreten y muestren este contenido de forma legible para los usuarios. A través de diversas etiquetas, HTML permite organizar el texto en diferentes secciones, como encabezados, párrafos, listas y enlaces.
A continuación tendrán ejemplos de los diferentes tipos de textos en HTML.

Párrafos

La etiqueta <p> se utiliza para definir un párrafo de texto. Un párrafo es una unidad de contenido que generalmente contiene varias oraciones relacionadas.

<p> Este es un párrafo de ejemplo. Contiene varias oraciones relacionadas que forman una idea completa. </p>

Encabezados (<h1> - <h6>)

Las etiquetas de encabezado (<h1> a <h6>) se utilizan para definir títulos y subtítulos en una página web. <h1> es el encabezado de mayor jerarquía y <h6> es el de menor jerarquía.

<h1> Título Principal </h1>
<h2> Subtítulo </h2>
<h3> Sección dentro del subtítulo </h3>

Textos destacados

Hay distintas formas de destacar el texto, las 3 principales son: texto en negrita, texto en cursiva y el texto marcado.

Para dejar el texto en negrita, se utiliza la etiqueta <b>
Para dejar el texto en cursiva, se utiliza la etiqueta <i>
Para dejar el texto marcado, se utiliza la etiqueta <mark>

<p> Este es un texto con una palabra en <b>negrita</b> para resaltar. </p>
<p> Me gusta leer <i>Don Quijote de la Mancha</i>, es un gran clásico. </p>
<p> Recuerda marcar las fechas importantes en tu calendario, como <mark>el 15 de noviembre</mark>. </p>

Extra

La etiqueta <hr> se utiliza para crear una línea horizontal que sirve como separación visual entre secciones de contenido.

<hr>

Ejemplo

Ahora que repasamos los diferentes tipos de textos que hay en HTML, vamos a dejar un ejemplo utilizando todos los conceptos nuevos aprendidos.

<!DOCTYPE html>
<html lang="es">
<head>
<title>Ejemplo</title>
</head>
<body>
<header>
<hr>
<h1>Hola mundo!</h1>
<hr>
</header>
<p>Esta es mi <b>primer</b> <mark>página web!!</mark></p>
<hr>
<footer>
<p><i>Todos los derechos reservados.</i></p>
<hr>
</footer>
</body>
</html>
