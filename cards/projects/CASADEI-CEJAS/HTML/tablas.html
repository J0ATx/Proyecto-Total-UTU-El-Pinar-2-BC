Tablas en HTML

¿Como crear tablas? ¿Como modificarlas?, etc.

Las tablas en HTML se utilizan para organizar y mostrar datos en filas y columnas de una manera estructurada. Son una herramienta muy útil cuando se necesita presentar información tabular, como por ejemplo, horarios, listas de productos , resultados de exámenes, estadísticas o cualquier otro tipo de dato que se pueda organizar en un formato de filas y columnas.

¿Cómo se crean tablas en HTML?

En HTML, las tablas se definen usando la etiqueta <table>. Dentro de esta, se utilizan varias etiquetas para crear las filas, las columnas y los encabezados de la tabla.
Estructura básica de una tabla:

<table>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th>
<th>Encabezado 3</th>
</tr>
<tr>
<td>Dato 1</td>
<td>Dato 2</td>
<td>Dato 3</td>
</tr>
<tr>
<td>Dato 4</td>
<td>Dato 5</td>
<td>Dato 6</td>
</tr>
</table>

Así se ve el código de ejemplo:

Encabezado 1 Encabezado 2 Encabezado 3
Dato 1 Dato 2 Dato 3
Dato 4 Dato 5 Dato 6

Etiquetas principales para las tablas:

<table>:

Define el contenedor de la tabla.

<tr> (table row):

Define una fila dentro de la tabla. Dentro de una fila se colocan las celdas de la tabla.

<th> (table header):

Define una celda de encabezado. Estas celdas se usan normalmente para describir los contenidos de una columna o fila. El texto dentro de un <th> generalmente se muestra en negrita y se centra de manera predeterminada.

<td> (table data):

Define una celda de datos dentro de una fila. Es la etiqueta más común y se usa para colocar la información o los valores dentro de la tabla.

Atributos comunes de las tablas:

border:

Define el grosor del borde de la tabla. Si no se especifica, por defecto las tablas no tienen borde.
Ejemplo:

<table border="1">

Así se vería una tabla con este atributo:

Encabezado 1 Encabezado 2 Encabezado 3
Dato 1 Dato 2 Dato 3
Dato 4 Dato 5 Dato 6

colspan:

El atributo colspan se utiliza para combinar varias celdas en una sola celda horizontalmente. Esto significa que una celda ocupará más de una columna. El valor de colspan especifica cuántas columnas debe abarcar la celda.
Ejemplo:

<table border="1">
<tr>
<th colspan="3">Encabezado combinado</th>
</tr>
<tr>
<td>Dato 1</td>
<td>Dato 2</td>
<td>Dato 3</td>
</tr>
<tr>
<td>Dato 4</td>
<td>Dato 5</td>
<td>Dato 6</td>
</tr>
</table>

Así se ve el código de ejemplo:

Encabezado combinado
Dato 1 Dato 2 Dato 3
Dato 4 Dato 5 Dato 6

rowspan:

El atributo rowspan se utiliza para combinar varias celdas en una sola celda verticalmente. Esto permite que una celda ocupe varias filas. El valor de rowspan especifica cuántas filas debe abarcar la celda.
Ejemplo:

<table border="1">
<tr>
<th colspan="3">Encabezado combinado</th>
</tr>
<tr>
<td rowspan="2">Dato combinado</td>
<td>Dato 1</td>
<td>Dato 2</td>
</tr>
<tr>
<td>Dato 3</td>
<td>Dato 4</td>
</tr>
</table>

Así se ve el código de ejemplo:

Dato
combinado		 Dato 1 Dato 2
Dato 3 Dato 4
